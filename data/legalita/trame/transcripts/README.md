# Transcript Trame

I transcript integrali non sono pubblicati nella pagina pubblica.

Naming previsto:

```txt
YYYY_trame_<video_id>_transcript.md
```

Front matter previsto:

```yaml
video_id:
title:
youtube_url:
edition_year:
event_date:
transcript_source:
transcript_status:
last_checked:
known_limits:
```

Corpo previsto:

```md
# Transcript

[00:00:00] Speaker unknown:
Testo...
```

Se lo speaker non e' certo, usare `Speaker 1`, `Speaker 2` o `Speaker unknown` e impostare `transcript_status: review_required`.
